# 2 V2ray Server
https://www.v2ray.com/

## 2.1 構築方法
- SSHでサーバーへ接続する

  OS：`Debian9`　か　`Centos7.5` 限定

- root権限取得

  `sodu -i`

- V2rayをインストール

  `bash <(curl -s -L https://git.io/v2ray.sh)`

>基本は全部デフォルト値でEnterキー連発

>Domainを持っている方は`Websocket＋TLS`がお勧め

>SSを配置する必要なし

- 接続情報を保存
